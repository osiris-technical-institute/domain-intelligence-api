## Long Description (for RapidAPI listing — use in "About" section)
The Domain Intelligence API gives you a complete picture of any domain in a single REST call. Whether you're building threat intelligence tools, onboarding verification flows, or competitive research dashboards — this API aggregates everything you need from authoritative sources.

### What You Get Per Domain:
- **WHOIS / RDAP** — Registrar, creation/expiry dates, registrant info via live RDAP queries with fallback chain
- **DNS Records** — A, AAAA, MX, NS, TXT, CNAME — full record sets with TTLs
- **SSL Certificate** — Issuer, expiry date, SANs, days remaining, validity status
- **Subdomains** — Passive enumeration via Certificate Transparency logs (crt.sh)
- **Email Security** — SPF, DKIM, DMARC policy analysis
- **Blacklist / DNSBL** — Real-time reputation check against major threat intelligence feeds
- **ASN / IP Info** — Autonomous system number, org, country, IP ranges

### Why Choose This API?
- ⚡ **Fast** — Redis caching layer, async lookups, p95 < 800ms for cached results
- 🔒 **Reliable** — RDAP fallback chain ensures WHOIS data even for tricky TLDs
- 📦 **Aggregate endpoint** — One call returns all data types, or use individual endpoints for what you need
- 💰 **Transparent pricing** — Free tier for prototyping, pay-as-you-grow plans
